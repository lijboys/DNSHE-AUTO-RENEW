# 🌐 DNSHE 免费域名自动续期助手 (CF Worker 精致版)

这是一个基于 Cloudflare Workers 的高级自动化脚本，专门用于批量自动续期 [DNSHE](https://my.dnshe.com/) 的免费域名。

本项目摒弃了传统的 GitHub Actions 部署方式，全程在 Cloudflare 网页版操作。它不仅能每月自动帮你续期域名，还**内置了一个精致的移动端 HTML 详情页**。当你收到微信推送时，点击卡片即可秒开排版优雅的运行报告，体验如丝般顺滑！

## ✨ 核心特性

- **☁️ 纯云端零成本**：完全依托 Cloudflare Workers，无需服务器，无需 GitHub 定时任务。
- **📦 多账号支持**：一行一个账号，支持无限挂载，一次运行扫平所有账号。
- **🛡️ 智能状态识别**：自动识别并跳过“未到续期时间”和“永久有效”的域名，只做有效请求。
- **📱 专属详情渲染**：独创的 `/detail` 路由解析，告别枯燥的纯文本，直接在浏览器渲染苹果风 (Apple-style) 的卡片报告。
- **🕹️ 防误触设计**：默认首页安全屏蔽，只有访问特有的 `/run` 路径才能手动触发续期，彻底告别“点开卡片就乱跑代码”的尴尬。
- **🔔 多渠道通知**：完美适配自建的微信 Push 后端以及 Telegram Bot。

## 🚀 极简部署指南

全程只需在 Cloudflare 网页版点点点，3 分钟即可完成部署！

### 第一步：创建 Worker 并填入代码
1. 登录 Cloudflare 控制台，进入 **Workers & Pages**。
2. 点击 **创建 (Create)** -> **创建 Worker**，随便起个名字（例如 `dnshe-renew`）并部署。
3. 进入该 Worker 的管理页面，点击右上角的 **编辑代码 (Edit code)**。
4. 清空里面的所有内容，把本项目中的完整 JavaScript 代码粘贴进去，点击 **保存并部署 (Save and Deploy)**。

### 第二步：配置环境变量 (Variables)
在当前 Worker 的管理页面，找到左侧菜单的 **设置 (Settings) -> 变量和机密 (Variables and Secrets)**。
在“环境变量”区域添加以下配置（**注意：类型请选择纯文本，方便以后随时查看和修改**）：

| 变量名 | 说明 | 格式示例 / 备注 | 是否必填 |
| :--- | :--- | :--- | :--- |
| `ACCOUNTS_CONFIG` | 多账号配置，**一行一个**，逗号隔开 | `key1,secret1`<br>`key2,secret2` | **必填** |
| `MY_URL` | 当前 Worker 的访问域名 | `https://你的前缀.workers.dev` | **必填** |
| `WX_API_URL` | 你的微信推送后端接口地址 | `https://push.xxx.dev` | 选填 |
| `WX_AUTH_KEY` | 你的微信推送专属鉴权密码 | `你的超复杂密码` | 选填 |
| `TG_BOT_TOKEN` | Telegram 机器人 Token | `123456:ABC...` | 选填 |
| `TG_CHAT_ID` | 接收 TG 通知的账号/群组 ID | `123456789` | 选填 |

> **💡 关于 `MY_URL` 的重要说明**：
> 请务必填写你这个 Worker 分配到的完整链接，不要带结尾的斜杠 `/`。这是生成“精美网页详情”卡片直链的核心！

### 第三步：设置每月自动运行 (Cron)
让它每个月自动打工：
1. 在 Worker 管理页左侧菜单找到 **触发器 (Triggers)**。
2. 往下拉找到 **Cron 触发器**，点击添加。
3. 选择 **自定义 Cron (Custom Cron)**，输入表达式：`0 2 1 * *` 
   *(代表每个月 1 号的 UTC 时间 2 点，即北京时间上午 10 点执行)*。
4. 保存即可。

## 🕹️ 如何使用与测试

部署完成后，你可以通过不同的 URL 路径来控制它：

- 访问 `https://你的MY_URL/` 
  👉 **默认首页**：显示绿灯运行状态，防止爬虫或外人误触。
- 访问 `https://你的MY_URL/run` 
  👉 **手动触发**：立刻执行一次全量续期，并在网页上直接打印结果，同时触发微信/TG推送。
- 微信卡片点击跳转
  👉 **精美详情页**：通过自带的 `/detail` 路由，瞬间渲染出本次续期的详细报表网页。
